# Timeline Exercise

A simple React app demonstrating the Timeline component from the `obvious` project at commit `5bcb87078`.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Features

- Interactive timeline with drag-and-drop reordering
- Visual date ranges with bars
- Single-day events displayed as point markers
- Today indicator
- Synchronized scrolling between task list and timeline grid
